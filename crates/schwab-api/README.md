# schwab-api

Rust client for the Charles Schwab **Trader API** (Accounts and Trading Production) — OAuth, accounts, orders, and transactions.

Part of [schwab-cli](https://crates.io/crates/schwab-cli) by [Soki Creative](https://soki-creative.com).
